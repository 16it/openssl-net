OpenSSL.NET

A managed OpenSSL wrapper written in C# for the 2.0 .NET Framework that currently exposes the crypto API. This a must for .NET developers that need crypto but don't want to use Microsoft's SSPI. This wrapper uses version 0.9.8g of libeay32.dll.

Download

The latest version can be downloaded here.

Installation

Make sure you have libeay32.dll in the current working directory of your application or in your PATH. In your .NET project, add a reference to the ManagedOpenSsl.dll assembly.
